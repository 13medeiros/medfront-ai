# Reflexo — PLAYFUL example

A tiny reflex toy: tap when the pad turns green; it measures your reaction time.
The **PLAYFUL** counterpart to the `service-booking` example — dark, saturated
and springy instead of calm and editorial.

## Run

Open [`src/index.html`](./src/index.html) in a browser. No server, no build, no
dependencies. Tap, click, or press <kbd>Space</kbd>.

## Why it's here

It proves MedFront AI produces **distinct identities per product**, not one house
style — verified with the convergence tool (see [`CASE_STUDY.md`](./CASE_STUDY.md)):
Reflexo vs the booking app scores **~37% — DISTINCT**.
