# Identity System — Reflexo

> PLAYFUL mode. Deliberately the opposite of a calm editorial UI: saturated flat
> blocks, chunky rounded shapes, springy motion.

## Core idea

**Central concept:** The screen is a **toy button**. The whole identity serves
one loop: anticipate → tap → react. Energy over calm; personality over restraint.

**Primary tension:** Stillness (waiting) vs. explosion (GO). The palette and
motion snap between the two.

## Visual grammar

**Geometry:** Big rounded rectangles (`--r: 28px`), one dominant shape (the pad).
Chunky offset shadow (`0 18px 0 -6px`) — a solid "physical button", not a soft
blur.

**Composition:** Centered, single-column, one hero target. Near-zero density.

## Typography

**Display + brand:** **Fredoka** (700) — rounded, friendly, loud.
**Body/UI:** system-ui — neutral, so the display carries the personality.
No mono: there is no "data to read", only a reaction time to celebrate.

## Color

| Token | Value | Role |
|---|---|---|
| `--base` | `#141b1a` | Canvas (dark, focused) |
| `--surface` | `#1e2a28` | Pad at rest |
| `--lime` | `#c3f53c` | **GO / success / brand accent** |
| `--pink` | `#ff5d8f` | **WAIT / hot** |
| `--yellow` | `#ffd23f` | **Too-early / attention** |
| `--text` | `#f4fff4` | Text on dark |
| `--ink` | `#0c110f` | Text on bright fills |

**Semantic-by-state:** each game state owns a color **and** an explicit label —
`waiting`→pink "Espere o verde…", `go`→lime "TOQUE!", `early`→yellow "Cedo
demais!". Never color alone.

## Verbal identity

**Tone:** Direct, punchy, warm-cheeky. "TOQUE!", "Cedo demais! 😅", "Relâmpago ⚡",
"Café primeiro? ☕".
**Avoid:** instructions-heavy copy, corporate calm, hype.

## Recognition test

Without a logo it still reads as *this* toy: dark base, one giant lime/pink pad,
Fredoka shouting a one-word command, a chunky button shadow. Nothing here would
fit the calm booking product — which is the point.

## Anti-patterns rejected

Purple-blue gradient, glassmorphism, tiny timid buttons, dense forms, decorative
blur. (Bounce/overshoot are *kept* — they are correct for PLAYFUL.)
